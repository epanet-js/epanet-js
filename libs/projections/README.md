# @epanet-js/projections

Projection domain for epanet-js: the `Projection` type (WGS84, XY-grid, proj4),
`WGS84` default, projection mappers, XY-grid transforms, ESRI WKT lookup, and
proj4-based coordinate transforms / GeoJSON CRS helpers.

Source-only workspace package (no build step) — consumed directly as TypeScript.
