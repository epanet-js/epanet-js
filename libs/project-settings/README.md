# @epanet-js/project-settings

Project settings domain for epanet-js: units, defaults, formatting, headloss
formula, and projection. Defines the `ProjectSettings` type, its defaults, the
per–unit-system presets, and the helpers for resolving decimals, minor-loss
units, and headloss defaults.

Source-only workspace package (no build step) — consumed directly as TypeScript.
